Module CWCfg

Addons Configurator global module

Usage:

    CWCfg:AddAddon('My Addon')
CWCfg:Set('My Addon','Print Hello',true,'bool',{category='Basic'})

Set (addonName, key, value, inputType, inpData)
Adds configurable input field to addon, according to Inputs:Fields function. Server function. Addon must be once added to configurer with AddAddon function.

Parameters:

  • addonName string name of the addon
  • key string configurable option name
  • value string default value
  • inputType string type of input, see Inputs:Create
  • inpData table additional data, like {category='Custom Category'}
